# CISC 360 — Computer Architecture

> **README generated:** 2026-02-22. This course was taken in 2000-2002 as part of the Computer Science program at the University of Delaware.

## Course Overview

Computer Architecture covers the design and organization of computer hardware — processor design, instruction set architectures, memory hierarchies, pipelining, and the relationship between hardware and low-level software.

Typical topics covered in CISC 360 at the University of Delaware include:

- Memory hierarchy: cache, RAM, virtual memory
