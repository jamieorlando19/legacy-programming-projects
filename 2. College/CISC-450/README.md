# CISC 450 — Computer Networks

> **README generated:** 2026-02-22. This course was taken in 2002-2003 as part of the Computer Science program at the University of Delaware.

## Course Overview

Computer Networks covers the theory and architecture of modern networking systems — the OSI and TCP/IP models, protocols at each layer (physical, data link, network, transport, application), routing algorithms, and network design principles.
